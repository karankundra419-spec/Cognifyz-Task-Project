# Cognifyz-Task-Project

 ## Project Overview

This project is a comprehensive data analysis case study on restaurant data, completed as part of the Cognifyz Data Analytics tasks. The objective of this project is to analyze restaurant trends, customer behavior, ratings, pricing, and service availability using Python-based data analysis techniques.

The analysis is structured into multiple levels (Level 1, Level 2, and Level 3), with increasing complexity, covering exploratory analysis, statistical insights, and relationship-based findings.

 ## Key Objectives

- Understand restaurant distribution across cities and cuisines
- Analyze ratings, votes, and pricing patterns
- Identify factors influencing customer ratings and popularity
- Explore relationships between price range, online delivery, and table booking
- Perform text analysis on customer reviews
- Detect restaurant chains and analyze their performance

 ## Tools & Technologies Used

- Python
- Pandas – data manipulation and cleaning
- NumPy – numerical operations
- Matplotlib & Seaborn – data visualization
- Text Analysis Techniques – keyword extraction from reviews

## Key Insights

- Certain cuisines and cuisine combinations consistently receive higher ratings
- Online delivery and table booking services positively influence ratings
- Higher-priced restaurants are more likely to offer premium services
- Popular restaurant chains show consistent customer engagement
